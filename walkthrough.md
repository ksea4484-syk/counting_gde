# 강동어울림복지관 게시글 통계 대시보드 V2 완료

요청하신 **V2(화이트 테마, 날짜 선택기, 최근 1주일 추이 차트, 특정 시트 연동)** 기능 구현이 모두 완료되었습니다!

## 1. V2 주요 변경 사항

### 🎨 세련된 화이트 테마 (Light Mode)
- 어두운 배경 대신 깨끗하고 모던한 **화이트(#f8f9fa, #ffffff)** 배경을 적용했습니다.
- 테두리(Border)와 부드러운 그림자(Shadow)를 활용해 카드가 살짝 떠 있는 듯한 세련된 카드 UI로 변경했습니다.
- 원형 차트를 가운데가 뚫린 도넛 차트(Doughnut Chart)로 변경하여 더 깔끔하게 보이도록 수정했습니다.

### 📅 날짜 선택 (Date Picker) & 전일 대비
- 상단에 특정 연/월/일을 선택할 수 있는 달력 입력 창이 추가되었습니다.
- 날짜를 변경하면 즉시 해당 일자의 게시글 수와 **전일 대비 증감량**으로 화면이 실시간 업데이트됩니다.
- 초기 로딩 시 시각적인 데모를 위해 2025년 1월부터 오늘까지의 가상의 임의 데이터를 생성하여 보여줍니다. (실제 구글 시트 연동 후 API가 연결되면 시트의 진짜 데이터를 가져옵니다.)

### 📈 최근 1주일 업로드 추이 차트
- 원형 그래프 우측에, 선택한 날짜를 기준으로 최근 7일(1주일) 동안의 총 업로드 수를 보여주는 막대 그래프(Bar Chart)를 추가했습니다.

### ⚙️ 구글 시트 특정 ID 연동 (Code.gs)
- 스크립트가 현재 열린 시트가 아닌, 요청하신 특정 시트(`11T84ToYo5kIaTPgNj9i_Xv8q1Q8Ef7AAGQTypo14YjY`)를 명시적으로 열어서 저장하고 불러오도록 백엔드 코드가 변경되었습니다.
- 프론트엔드에서 날짜 필터링과 7일 추이 차트를 그릴 수 있도록, `doGet` API가 당월 데이터뿐만 아니라 시트에 있는 모든 전체 날짜 데이터를 JSON으로 응답하도록 수정되었습니다.

## 3. 과거 데이터 입력 안내 (2025년 1월~)

구글 앱스 스크립트(GAS)는 스크립트가 동작하는 시점(오늘)부터 크롤링을 시작하므로, 크롤링 대상 사이트가 과거 기록을 모두 제공하지 않는 이상 2025년 1월 데이터가 자동으로 쌓이진 않습니다.
2025년 1월부터의 조회를 원하신다면 아래 방법을 사용해 주세요:

1. 연동하신 구글 시트(`11T84ToYo5kIaTPgNj9i_Xv8q1Q8Ef7AAGQTypo14YjY`)의 `Data` 시트를 엽니다.
2. A열(Date)부터 E열(YouTube)까지, 2025년 1월 1일 날짜부터 과거 데이터를 엑셀 타이핑하듯 수동으로 직접 채워 넣어 주시면 됩니다.
   - 포맷: `2025-01-01` | `5` | `2` | `1` | `0`
3. 시트에 데이터를 입력해 두면, 대시보드의 달력(Date Picker)에서 2025년 1월 날짜를 선택했을 때 정상적으로 해당 수치와 그래프가 연동되어 표시됩니다.
